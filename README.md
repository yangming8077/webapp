## 运行命令
```javascript
npm install 		//安装插件
npm start 	        //启动开发环境

```
## 文件目录
| **目录\文件名称**      |    **说明**   | 
| :-------- | :--------|
| app  | 项目工程开发目录 | 
| build | 项目生成目录 | 
| postcss.config.js | css文件优化配置 | 
| package.json | 项目应用插件管理 | 
| webpack.js|  开发配置文件 |
